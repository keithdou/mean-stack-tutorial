MEAN Development Stack Tutorial

Introduction

Pre-requisites

You will need to install a Node.js development environment. The easiest way to install NodeJs is by downloading the appropriate installer for your platform. This will also install the Node Package Manager(npm). Sometimes, the npm version bundled with Node is not the latest version, but for the purposes of this tutorial the bundled version will be sufficient. You will also need to install a MongoDb database.

Features

  • Server-side Javscript Web Service using Express
  • Persistence layer managed by a NoSql database using MongoDb and mongoose
  • Client authentication using Json Web Tokens (JWT)
  • Securing passwords using Nodejs crypto
  • Server-side API role restrictions using express-permissions
  • Server-side field validation using express-validator

Project Setup

Packages

Now that you have created your project, change directory to the project directory and install the following npm packages: 
npm install mongoose;
npm install express-validator;
npm install express-jwt;
npm install express-validator;
npm install jsonwebtoken;
npm install morgan;
npm install morgan-body;
